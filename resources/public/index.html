Guide

1. Create a Slash command

Create a Slash command with the following attributes:

Command: /meme
URL: https://slacky-server.herokuapp.com:443/api/slack/meme
Method: POST
Autocomplete description: Create a meme
Autocomplete usage hint: image url or search term | upper text | lower text

Take note of the token. You will use this to register your account.

2. Create an incoming webhook

Create an incoming webhook.

Take note of the webhook url. This is the second piece of information needed to register your account.

3. Register your account

Fill in and submit the form using the information you have collected.

4. Create all the memes!

Type /meme into your Slack and never look back! See the examples below.
